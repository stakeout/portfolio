# Проект "Портфолио"
**проект с курса продвинутой веб-разработки Loftschool 19.09.2016**


## Старт проекта

### Склонируй репозиторий и перейди в папку проекта
```
git clone git@github.com:stakeout/portfolio.git new-project && cd new-project
```

### Установи модули
```
npm i
```

### Запусти шаблон
```
npm start
```
